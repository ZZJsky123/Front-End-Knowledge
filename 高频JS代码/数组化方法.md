## arguments 数组化方法

```
arguments是一个对象，程序书中有写它是类数组，那什么是类数组

arguments = {0：value1,1: value2, 2: value3, length:3}

数组化方法：
     1. var args = [].slice.call(arguments);
     2. var args = Array.prototype.slice.call(arguments);
     3. var args = [...arguments];
     4. var args = Array.apply(null, arguments)
     5. var args = Array.from(arguments)
```
