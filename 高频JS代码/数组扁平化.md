## 数组扁平化方法

```javascript
扁平化： 作为新的企业管理方式提出，从前企业总体的管理方式： 高层-中层-基层，在实施过程高层到中层的链路冗长，使得决策向下传输时
        执行效率低，高层和基层分割严重，基于此背景提出扁平化削减中间层级，高层可以和基层形成流畅的沟通
        
let a = [2, [2,3,[1,2,3]]];

数组扁平化：将中间的多维数组（嵌套数组）进行展开，使得多维数组最终展开成为一维数组

function flatten(arr){
    if(!Array.isArray(arr)){
       throw new Error('')
    }
    
    return _flatten(arr);
    
    function _flatten(arr){
        
       let res = [];
    
        for(let i of arr){
            if(Array.isArray(i)){
                let temp = _flatten(arr);
                res.push(...temp);
            }else{
                res.push(i);
            }
        }
        
       return res;
    } 
}
```
