# px

```
1. 像素点单位，相对单位，其相对于显示器屏幕分辨率而言
```



## em

```
1. 相对长度单位，其相对于当前文本的字体尺寸，若当前文本内没有设置
   字体尺寸，则根据浏览器默认尺寸 1em = 16px
2. 可在body选择器声明中：font-size:62.5%,
   如此设置： 1em=10px，1.6em = 16px， em到px的换算乘10即可
3. 因为em是相对于当前文本设置，当父元素设置1.2em 子元素设置1.2em
   则子元素的实际大小是1.2em*1.2em=1.44em
```

## rem

```
1. 相对长度单位，相对于网页根元素html来设定的
   使用rem不存在由于元素嵌套而导致的字体混乱
2. 1rem = 16px  12px = 0.75em
3. 为了使用方便设置 html{foot-size:62.5%}  16*62.5% = 10px
   1rem = 10px 1.2rem = 12px
```
