### CSS 基础

```
1.  HTML 世界中的 CSS
  a. HTML 中任何内容都是一种内容都是一种节点， 在HTML中显示的是三种节点：
                   文本节点、 注释节点、 元素标签节点
     CSS 通过 [盒模型] 施加对三种节点添加样式： 位置布局， 内容显示
     [位置布局]： 页面生成后， 标签在页面中的位置显示
     [内容显示]： 页面生成后， 标签中的内容
     
  b. HTML 元素在 CSS 眼中都是盒子， 其结构是：margin、 border、 padding、 content，
     
     I. 由于 display: incline-block 属性， 因此每个元素其是两层盒子， 外面的盒子被称作
        外在盒子，两种类型： 块级盒子 和 内联盒子。
     II. 文字又被称为匿名内联元素，它的盒子： em框， 内容区， 行内框， 行框。
     III. 内联元素之前都有一个空白的幽灵节点，宽度和高度都为 0。
     
2. HTML 中的环境
     a. HTML 中高度是无限的， 宽度是有限的， 因此 HTML 基本的环境是流， 其决定了
        元素在水平方向的摆放方式。
     
     b. 流： 按照每行元素个数和已知元素宽度，自动使所有元素铺满整个容器宽度。
         鑫三无准则的一条： 无宽度， 不规定元素宽度，是元素使用流进行布局
         
     c. BFC: 块级格式化上下文， 当添加特定属性后， 父类标签会成为块级格式化上下文
        里面元素独立于外面元素， 不相互影响。通过 BFC 我们可以实现更加智能的自适应
        布局。 
     
     
```
