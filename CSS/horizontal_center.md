## 水平居中

```
定义： 元素左右间距一样

行内元素水平居中的方式：
   text-align：center     (text-align: 在一条水平线上的文本)
   
块级元素水平居中方式：
   absolute+margin:0 auto          (上下0 左右auto)
   absolute+translate      设置absolute { left:0  right:0}

使用flex布局：
   display：flex           (flex: 平铺)
   justify-content:center;
```

## 垂直居中

```
定义：元素在所在区域上下间距相同
```
